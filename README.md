# Create JWK to files

Just a short script to create new JWKs and write out to files for transportation.

## Run
```$ create-jwk.py -o myJWK```

1. Generates keys.
2. Pretty prints them to stdout.
3. Writes to files.

## Outputs
```
myJWK_full_<date>.json
myJWK_private_<date>.json
myJWK_public_<date>.json
```
